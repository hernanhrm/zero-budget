# Plan: Replace Hardcoded Colors with CSS Variables

## Summary
Replace all hardcoded hex colors in the members feature with CSS variables defined in the design system.

## Hardcoded Colors to Replace

### `loading-row.tsx`:
- `bg-[#2D2D2D]` → use `--skeleton` variable

### `members-table.tsx`:
- `bg-[#2D2D2D]` → use `--skeleton` variable
- `text-[#1A1A1A]` → use `--foreground` (dark on primary)
- `text-[#6B6B6B]` → use `text-muted-foreground`
- `text-[#F5F5F0]` → use `text-foreground`
- `text-[#3D3D3D]` → use `--muted-foreground` or new variable
- `text-[#FF6B35]` → use `text-destructive`

## Files to Modify

### 1. Add CSS Variables to `globals.css`
**Path**: `frontend/packages/ui/src/styles/globals.css`

Add skeleton color variable in both `.light` and `:root`:
```css
--skeleton: oklch(0.269 0 0);  /* #2D2D2D in dark mode */
```

And add `muted-secondary` for the darker muted text:
```css
--muted-secondary: oklch(0.269 0 0);
```

### 2. Update `loading-row.tsx`
**Path**: `frontend/apps/app/src/features/members/components/loading-row.tsx`
- Replace `bg-[#2D2D2D]` with `bg-skeleton` or `bg-[var(--skeleton)]`

### 3. Update `members-table.tsx`
**Path**: `frontend/apps/app/src/features/members/components/members-table.tsx`
- Replace all hardcoded colors with appropriate CSS variables
